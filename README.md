

The code of paper [DiffMissing: Denoising Diffusion Model for Multivariate Time Series Forecasting with Variable Missing](https://ieeexplore.ieee.org/document/11210227)



### Datasets

- METRLA
- PEMS08
- PEMSBAY
- Weather

### How to run

```bash
cd scripts/
bash ./METRLA.sh
bash ./PEMS08.sh
bash ./PEMSBAY.sh
bash ./Weather.sh
```
